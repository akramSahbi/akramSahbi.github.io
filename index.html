Arezzo Designer

JAVA DEVELOPMENT

I conceived a NetBeans Platform Application designer of industrial networks using Java and NetBeans Visual Library.

The project consists of creating a network of industrial devices which consists of workstations, switch components, and sensors from a palette, linking them together and adding shuttles on top of the links that transport products.

Then Configure the devices and validate the input, test the configuration for its conciseness (closed network, switch components are linked), and serach for loops and colorize them, and finally export the network definition files to be deployed at the Arezzo Emulator. That will take care of animating the industrial network created using the designer.

Arezzo designer helped me learn about Widget design pattern and modular programming using Object Oriented Programming JAVA and the open source NetBeans Platform.

ArezzoDesigner also initilized me to multidisciplinary research; since this project is an application on Information Technology on Industry.
